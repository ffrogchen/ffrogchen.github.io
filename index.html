HEXO博客搭建

一开始改 node_modules/hexo-server/index.js 的端口配置，其实不一定有必要去改。

必须进入blog目录也就是你的那些hexo文件配置文件在的地方，否则localhost:4000就会失败。
我是在管理员启动dos,然后进入文件目录启动才成功，汗

打开站点的配置文件_config.yml，翻到最后修改为：
deploy:
type: git
repo: https://github.com/ffrogchen/ffrogchen.github.io
branch: master
输入命令时冒号啥的是英文冒号，中文冒号出问题，要开英文键盘写命令

deploy之后，ssh登录不保存钥匙会就向你要用户名密码

改完配置，必须hexo clean
hexo g
hexo d

HEXO博客搭建参考：https://zhuanlan.zhihu.com/p/26625249
git的一些基础命令：https://www.cnblogs.com/libin-1/p/5918468.html
Hexo-修改Hexo主题：https://www.jianshu.com/p/33bc0a0a6e90
